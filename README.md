<h1>Fruit Fly Pool</h1>

<hr>

Hi there!<br><br>
This is my first JS game project.<br>
Have fun, and good luck!

<h3>Keys:</h3>
* 'W' : Increase power.<br>
* 'S' : Decrease power.<br>
* 'Esc' : Return to menu.<br>

<h3>Mouse:</h3>
* click : shoot <br>
* Aim by moving mouse<br>
